# Les conversions de type 

Quels sont les résultats de ces expressions ?

```js no-beautify
"" + 1 + 0      // (a)
"" - 1 + 0      // (b)
true + false    // (c)
6 / "3"         // (d)
"2" * "3"       // (e)
4 + 5 + "px"    // (f)
"$" + 4 + 5     // (g)
"4" - 2         // (h)
"4px" - 2       // (i)
"  -9  " + 5    // (j)
"  -9  " - 5    // (k)
null + 1        // (l)
undefined + 1   // (m)
" \t \n" - 2    // (n)
```

Réfléchissez bien, notez et comparez avec la réponse.
